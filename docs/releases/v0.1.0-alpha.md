# Release v0.1.0-alpha

## Highlights

- Polished Basar public identity and Apache-2.0 software licensing.
- Static browser MVP with a built-in demo source, local evidence search, local fallback answers, claim cards, and exportable 0G growth packages.
- Local-first API (FastAPI) with Source Cards and Claim Cards.
- Deterministic source scoring and local retrieval (SQLite + FTS5 fallback path).
- Mock AI answer flow with cited responses and no-hallucination fallback.
- CLI for init, doctor, status, demo, ingest, search, score, ask, export, and 0G archive dry-run.
- Browser-local frontend (Vite) for browse, ask, user-owned 0G Router settings, and 0G growth packages.
- 0G-first public web direction with env-driven provider support for local and self-hosted deployments.
- Open-source governance: license, governance, security, and community handoff docs.

## 0G and Sharing Notice

- 0G participation is user-owned: users bring their own Router, storage workflow, and credentials.
- Network-shared docs avoid hardcoded localhost or machine-local paths.
- Local files and archives remain in local data paths by default.
- Public web builds do not depend on the maintainer's machine or shared backend credentials.

## Data Governance

- No copyrighted books or paywalled corpora in sample/repo data.
- No secrets or private user archives committed.
- Public examples are CC0/self-authored only.
- Local coordination notes are excluded from release packages.
