# Release v0.1.0-alpha

## Highlights

- Local-first API (FastAPI) with Source Cards and Claim Cards.
- Deterministic source scoring and local retrieval (SQLite + FTS5 fallback path).
- Mock AI answer flow with cited responses and no-hallucination fallback.
- CLI for init, ingest, score, ask, export, and 0G archive dry-run.
- Optional frontend (Vite) for browse, ask, and 0G status.
- 0G providers as secure scaffolds (dry-run only, env-driven config).
- Open-source governance: license, governance, security, community handoff docs.

## 0G and Sharing Notice

- 0G integration is optional and scaffolded for dry-run in this release.
- Network-shared docs avoid hardcoded localhost or machine-local paths.
- Local files and archives remain in local data paths by default.

## Data Governance

- No copyrighted books or paywalled corpora in sample/repo data.
- No secrets or private user archives committed.
- Public examples are CC0/self-authored only.
