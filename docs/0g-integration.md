# 0G Integration

0G is optional. The local MVP works without 0G.

Intended use cases include decentralized backup, AI compute provider support, and future fine-tuning workflows. Credentials must be environment variables. Dry-run mode exists for storage and compute scaffolds.

Before production use, developers must verify current official 0G SDK/API docs and replace TODOs in the 0G providers. Do not invent SDK calls or commit secrets.
