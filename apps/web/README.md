# Basar Web

Static Vite UI for 0G-first, browser-local Basar workflows.

```bash
npm install
npm run dev
```

The public build does not call a project backend or `localhost`. Users can enter
their own 0G Router endpoint, model, API key, and optional 0G Storage compatible
endpoint in Settings. Source data is stored in the user's browser storage by
default and can be exported or published as a 0G growth package.
